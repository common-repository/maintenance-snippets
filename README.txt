This plugin gives you a report of plugins that need updates.

== Description ==

This plugin gives you a report of plugins that are not updated. Then you can copy and paste the old version plus the new version to send to your clients telling them what you've done for your maintenance work.

== Installation ==

The process to install this plugin is the default one:

1. Upload maintenance-snippets folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
